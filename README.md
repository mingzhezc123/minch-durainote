# 榴莲记事

## [在线demo](http://www.minzer.top/durianote/index.html)

## 介绍

> * 这是一个可运行在web端和移动端的todolist类应用
> * 具备添加和删除记事项以及更改记事项状态的功能
> * 该应用由Hbuilder打包，已有.apk和.ipa(还不能直接使用)文件，可安装到移动端

## 技术栈

> H5 + C3 + js + localstorage + svg
